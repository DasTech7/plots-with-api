# Plots And Api App

see the title

### How To Run
1. Install the library with this command
```bash
$ pip install -r requirements.txt
```
2. Run app with
  * using api (for api only)
  ```bash
  $ uvicorn src.app:app --reload
  ```
  * Or with `matplotlib`
  ```bash
  $ cd src/
  $ python3 plot.py || python plot.py
  ```


> enjoy :)
